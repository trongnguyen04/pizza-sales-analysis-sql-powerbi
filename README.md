# Pizza Sales Analysis using SQL and Power BI

## Project Overview

This project analyzes pizza sales data to evaluate business performance,
identify sales trends, and determine the best-selling and worst-selling pizzas.

SQL Server was used for data analysis, while Power BI was used to build
an interactive dashboard.

## Tools Used

- SQL Server
- SQL Server Management Studio
- Power BI
- Power Query
- DAX

## Business Questions

- What is the total revenue?
- How many orders were placed?
- How many pizzas were sold?
- What is the average order value?
- Which days and months have the highest number of orders?
- Which pizza categories and sizes generate the most revenue?
- Which pizzas perform best and worst?

## Dashboard Preview

![Pizza Sales Dashboard](dashboard-overview.png)

## Key Insights

- Friday and Saturday recorded high order activity.
- The Classic category made a strong contribution to sales.
- Large pizzas generated the highest revenue share.
- Thai Chicken Pizza was a key revenue driver.
- Classic Deluxe Pizza led in quantity sold and order frequency.

## Recommendations

- Prepare additional staff and ingredients during peak periods.
- Maintain sufficient inventory for popular pizzas.
- Use high-volume products in promotional bundles.
- Review underperforming products using revenue, quantity, cost, and profit.
